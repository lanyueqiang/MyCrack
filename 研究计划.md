# 机场跑道裂缝检测论文研究计划（确定版）

## 一、选题定位

**题目方向**：基于物理先验驱动的机场跑道裂缝语义分割方法  
**网络名称（暂定）**：CrackPriorNet  
**目标期刊**：SCI 一区/二区（如 Automation in Construction, Engineering Applications of AI 等）  
**数据集**：自采集机场跑道裂缝图像，2000张，分辨率 1800×900，已有像素级标注

**核心叙事**：不同于现有方法将裂缝检测视为通用语义分割问题，本文深入分析裂缝的
物理成像机理（光照遮蔽→灰度谷效应）和形态学特征（方向性、连续性），将领域先验
嵌入到特征提取、卷积操作和损失函数中，构建物理先验驱动的裂缝分割框架。

## 二、创新点方案（已确认）

### 创新点1：灰度谷感知特征增强模块（Valley-Aware Feature Enhancement, VAFE）
- **层面**：特征提取
- **原理**：裂缝因光照遮蔽，灰度横截面呈谷形——中心最暗、边缘过渡
- **方法**：多方向Hessian谷响应提取 + 与编码器浅层特征融合
- **优势**：显式利用物理先验，区分裂缝与其他暗色干扰

### 创新点2：方向引导可变形卷积（Direction-Guided Deformable Convolution, DGDC）
- **层面**：卷积操作
- **原理**：裂缝具有局部方向一致性，标准方形卷积核不适配细长曲线结构
- **方法**：轻量方向预测头估计走向角度θ → 引导可变形卷积偏移量沿裂缝方向排列
- **优势**：卷积核自适应沿裂缝走向采样，增强连续性感知

### 创新点3：灰度先验加权损失函数（Intensity-Prior Weighted Loss, IPWL）
- **层面**：损失函数
- **原理**：裂缝中心比边缘更暗，利用这种灰度梯度为不同像素分配不同学习权重
- **方法**：结合归一化灰度值 + 到裂缝骨架的距离变换 → 生成像素级权重图
- **优势**：引导网络先学确定性高的暗线核心，再扩展到模糊边缘

### 后续工作方向：
- 物理驱动裂缝数据增强（PCA）：基于形态规则程序化生成合成裂缝

## 三、实验方案

> 详见 [实验方案设计.md](实验方案设计.md) 完整版

### 3.1 数据集
- **主数据集**：ARCD（自采集，2000张），70/15/15划分
- **公开数据集**：Crack500、DeepCrack、CFD（泛化性验证）

### 3.2 对比方法（8个）
- 通用分割：U-Net、DeepLabV3+、HRNet、SegFormer
- 裂缝专用：DeepCrack、FPHBN、CrackFormer、CT-CrackSeg

### 3.3 评价指标
- 核心：Precision / Recall / F1 / mIoU / IoU_crack
- 辅助：ODS / OIS / AUC / Params / FLOPs / FPS

### 3.4 消融实验
- 核心消融：M0~M7（3模块的全组合，8组实验）
- 模块内部消融：VAFE(方向数/融合位置)、DGDC(引导方式/版本/位置)、IPWL(各成分贡献)
- 超参敏感性：α, β, Hessian σ, 采样点数

### 3.5 额外分析
- 可视化（分割结果、特征图、采样点、权重图、Grad-CAM）
- 效率分析（精度-效率散点图）
- 裂缝类型细分评估
- 泛化性实验（跨数据集零样本测试 + Crack500重新训练对比）
